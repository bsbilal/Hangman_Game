# Hangman - (Adam Asmaca)

Hangman is a simple word guessing game. Players try to figure out an unknown word by guessing letters. If too many letters which do not appear in the word are guessed, the player is hanged (and loses).

## Getting Started

Play the game by pressing 'new game button' (Words are took from website by JSOUP so words are always be updated). As letters in the word are guessed, write them above the cooresponding underline.
